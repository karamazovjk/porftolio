### welcome to my project!

The objective of this porfolio is to carry out a sequence of analyzes with data and studies on sl stress and screen time,eep, and how all of this correlates using exploratory data analysis and visualizations to identify possible patterns. To investigate whether there is a correlation between daily screen time and increased stress, as well as possible impacts on sleep duration.

Key Questions
- Does higher screen time correlate with increased stress?
- Is sleep duration negatively affected by excessive smartphone usage?

Methods
- Data cleaning and exploratory analysis in Python
- SQL queries for aggregation and pattern detection
- Visualization in Power BI
